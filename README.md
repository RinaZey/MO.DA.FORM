#### Задачи реализуемые платформой 

- [ ] Добавление актуальной информации для одногрупников 

Идеи для реализации:

1) Раздел новости, в котором будет актуальная информация в каждом из подразделов:
   1.1) Новости про олимпиады, хакатоны, финатлон, написание научных статей и работ и т.д.
   1.2) Новости про стажировки и практики от компаний-сотрудников с политехом
   1.3) Иные актуальные новости вуза (уведомление об отмене или переносе пар (-ы) (также в связи с праздничными днями), уведомление о переходе на другую платформу в связи с неработоспособностью или отсутствием ссылки вебинар и т.д.

2) Раздел домашние задания
В данном разделе будет находится вся актуальная домашняя работа, которую необходимо выполнить обучающемуся, для допуска к экзаменам или получения оценки-автомат.

3) Раздел сроки сдачи работ
В данном разделе будет находится актуальная информация о сроке сдачи работ.

4) Раздел информации о преподователях
В данном разделе будет находится вся актуальная информация о преподователях (ФИО, контактная информация, расписание), для того, чтобы обучающимся было легко найти способ как связаться с интересующим его преподавателем.

5) Раздел с ссылками на другие платформы вуза (СДО, ЛК, группу вк, телеграмм)
